# Múltiplo de sete

![_](assets/cover.jpg)

Implemente um programa que recebe um número inteiro e imprime "SIM" caso ele seja múltiplo de 7, e "NAO" caso contrário.

### Entrada

- Um número inteiro.

### Saída

- "SIM" caso o número seja múltiplo de 7.
- "NAO" caso o número não seja múltiplo de 7.
