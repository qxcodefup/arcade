# Contar repetidos

![_](assets/cover.jpg)

### Entrada

- 1a linha: O número P(Procurado) e N(Tamanho do vetor).
- Nas próximas N linhas, cada linha contém um número inteiro representando um elemento do vetor.

### Saída

- A quantidade de vezes que o número P aparece no vetor.

## Testes

``` py
>>>>>>>> INSERT
5 3
2
3
6
======== EXPECT
0
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
0 4
0
0
1
0
======== EXPECT

<<<<<<<< FINISH
```
