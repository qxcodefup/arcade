# Calculando se é

![_](assets/cover.jpg)

## Contexto

Dado um número inteiro, faça uma função recursiva que verifique se ele é primo.  

### Entrada

- Um número inteiro **N**.

### Saida

- O número **1** se **N** for primo e **0** caso contrário.

## Testes

``` py
>>>>>>>> INSERT
5
======== EXPECT
1
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
21
======== EXPECT
0
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
2
======== EXPECT
1
<<<<<<<< FINISH
```
