# Calcule o mmc

![_](assets/cover.jpg)

Faça uma função que calcule e retorne o mmc de dois números.  

### Entrada

* Dois numeros inteiros N e M.

### Saida

* O mmc de N e M.

## Exemplos

<!-- load tests.toml --tests 2 -->
```py
>>>>>>>> INSERT
6 9
======== EXPECT
18
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
3 4
======== EXPECT
12
<<<<<<<< FINISH
```
