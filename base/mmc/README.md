# Calcule o mmc

![_](assets/cover.jpg)

## Ação

Faça uma função que calcule e retorne o mmc de dois números.  

### Entrada

* Dois numeros inteiros N e M.

### Saida

* O mmc de N e M.

## Testes

``` py
>>>>>>>> INSERT
6 9
======== EXPECT
18
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
3 4
======== EXPECT
12
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT
3 6
======== EXPECT
6
<<<<<<<< FINISH
```
