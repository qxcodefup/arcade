# Calculando se é

![_](assets/cover.jpg)

## Contexto

Dado um número inteiro, faça uma função que verifique se ele é primo.  

### Estratégias

- Faça um laço de i = 2 até sqrt(n) para ver se o valor é divisível por i

### Entrada

- Um número inteiro **N**.

### Saida

- O número **1** se **N** for primo e **0** caso contrário.

## Testes

``` py
>>>>>>>> INSERT
5
======== EXPECT
1
<<<<<<<< FINISH
```

