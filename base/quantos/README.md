# Quantos são iguais

![_](assets/cover.jpg)

## Contexto

Implemente um programa que receba três valores e informe quantos deles são iguais.

### Entrada

- Três números inteiros, um por linha.

### Saída

- Um número inteiro indicando quantos valores são iguais dentre os três fornecidos.
